# Paraşüt Ayar Notları

Script Properties içinde tutulması gereken alanlar:

- `PARASUT_CLIENT_ID`
- `PARASUT_CLIENT_SECRET`
- `PARASUT_COMPANY_ID`
- `PARASUT_ACCESS_TOKEN`
- `PARASUT_REFRESH_TOKEN`
- `PARASUT_ACCESS_TOKEN_EXPIRES_AT`
- `PARASUT_CALLBACK_URL`
- `PARASUT_PRODUCT_ID_MAP_JSON`
- `PARASUT_CONTACT_ID_MAP_JSON`

Değerler GitHub'a yazılmaz. Bu repo yalnız alan adlarını ve örnek JSON şablonlarını içerir.
